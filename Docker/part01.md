# Docker 安装
首先先去[官网](https://docs.docker.com/docker-for-mac/install/#download-docker-for-mac)下载docker客户端

安装完成后，需要登录docker客户端
![安装完成图片](https://github.com/it037218/Document/blob/master/Images/001.jpeg?raw=true)

在终端中输入``docker version``
